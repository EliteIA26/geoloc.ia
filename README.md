# geoloc.ia — Panel Territorial Agrícola de La Rioja

Cockpit territorial que transforma dados de satélite gratuitos (Sentinel-2) em um painel
de decisão para o governo provincial de La Rioja, com prévia da visão do produtor.

Demo F0 navegável — instrumento de venda/validação para o canal CFI / Ministerio de Producción.
